# 💫 About Me

I build with AI.

Mobile apps, full-stack web apps, desktop software — if it needs to exist, I can probably build it.

I adapt to whatever stack fits the project:

* Flutter / Expo for mobile
* Next.js / React for the web
* Rust when performance actually matters
* Supabase for most backend work

I also train ML models and experiment with local AI systems.

I treat AI like a real development tool, not a gimmick.
